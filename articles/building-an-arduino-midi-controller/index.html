what's the difference between arduino models

- Arduino boards with native USB support: Arduino Due, Arduino Zero, Arduino Leonardo, Arduino Micro, Arduino LilyPad USB ...

- Some of the newer boards have native USB support (i.e. the USB connection goes directly to the main MCU) this means that they can pretend to be a USB MIDI device. This is based on PluggableUSB, using the MIDIUSB library. All you have to do to get it working, is installing the library, and upload a sketch. The computer will then automatically detect the Arduino as a MIDI
