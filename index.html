I write Code

Whether there's a new Web standard arriving soon on a specific version of a specific browser or I had a spontaneous idea on the tram, I always try to create a quick demo for myself. That helps me understand better what the possibilities are.

  1. CSS 3D Sphere, 2021

    21 stacked bordered elements separated by equal distances create perception of volume.

  2. Spell checker, 2020

    After counting words from 20 popular books, it suggests the nearest word to replace spelling mistakes.

  3. Wunderpaint, 2020

    Pixel Art editor that uses an HTML canvas only to process input/output. The user interacts with a base64 image.

  4. Punchy, 2020

    WebMIDI and WebAudio allows playing a MIDI controller on a synthesizer or sampler directly in the browser.

  5. Feeds Aggregator, 2019

    Reads RSS feeds using a serverless function and normalizes the output to be displayed in a list.

  6. CSS Voxels Chicken, 2019

    Each voxel is one HTML element with 3 faces. Voxels are grouped and stacked in layers on the z-axis.

  7. CSS Voxel Composition

    Feb, 2019

  8. Single-Element Cube, 2019

    The element and pseudo-elements are the 3 faces. When it rotates, it changes the transform origin.

  9. Single-Element Logo, 2018

    Playing with shapes as borders, backgrounds, and characters as part of the content of the pseudo-elements.

  10. CSS Calculator, 2018

    A fully working CSS Casio calculator built with Create React App and reducers to update the state.

  11. Filters for Humans, 2018

    Conceptual experiment to minimize the users' learning curve for filtering lists.

  12. Autumn Leaf, 2018

    An emoji flattened with the text-shadow technique, overrotating in a bunch of axes.

  13. Mobile Game Patterns, 2013

    Collection of mobile games design patterns for the inspiration of developers, designers, and artists.

  14. Rounded Bar, 2012

    Handling an SVG path from one single variable to show a wheel indicator.

  15. Time is Ticking, 2012

    Modifying the new CSS property transform-origin to distribute letters around a circle.

  16. Doodle Lander, 2012

    Playing with artificial gravity on canvas to recreate some of the physics of Moon Lander in JavaScript.

  17. 70 Sec. of Terror, 2012

    Interactive infographic about the landing of Curiosity Mars Rover, in 13 kB including vector graphics.

  18. Doodle Car, 2012

    My first experiment using canvas. It plays with forces like acceleration, inertia, momentum, etc.

  19. Isometric World, 2011

    DOM-based matrix of tiles. When clicking on them, it calculates the shortest path and moves the player there.

  20. SkiFree DOM, 2011

    Recreation of the game for Windows using HTML elements handled via JavaScript.

  21. Pick a fruit!, 2009

    My first fully working experiment. It required so much processing to walk that the pace depended on the browser.

I make Things

These contraptions are a mix between open-source hardware and software. I film myself building them to encourage others. When it comes to publishing videos, I don't follow a schedule. I believe the schedule pressure compromises quality, and I prefer to create things I genuinely enjoy.

  1. Game Console, 2022

    The best electronics for game emulation inside a 3D printed case I designed using the xbox elite controller as a reference.

  2. Simpsons TV, 2021

    Inspired by Brandon Withrow's project, a Raspberry Pi plays (offline) random episodes 24/7 on an old CRT TV.

  3. 3D Printed RC Car, 2021

    Fully-printed, 4WD, high-performance, 1:8 scale RC car. Even the gears are fully printed in PLA, and they work smoothly.

  4. Word Clock, 2021

    I found a vintage clock on the street and upcycled it into a word clock using an individually addressable LED strip and Arduino.

  5. Tesla Roadster, 2020

    3D printed parts are rough, and I found the perfect project to learn as much as possible about how to post-process them.

  6. MIDI Controller, 2020

    I built this as a minimum viable product for beginners. It has no knobs, sliders, or faders, and it runs with minimal code.

  7. UV Light Box, 2020

    This simple box can make cyanotypes, a photographic printing process that produces blueprints using artificial sunlight.

  8. Arcade Machine, 2018

    I put together an old Raspberry Pi, monitor, and speakers with MDF boards, Sanwa-like joysticks, and HAPP push buttons.

I draw Pixel Art

This medium challenges me to work within a set of hard constraints, like using an extremely reduced amount of space or colors. So, I play with Pixel Art not necessarily to evoke nostalgia but for the art of abstraction.

  1. Weltballon, 2020

    Another one on my list of "odd things that I've never seen before that call my attention every time I pass by". I prefer working with less colors and size, but I wanted to include everything.

  2. Wasserturm Ostkreuz, 2020

    I'm lucky to see this historical one-hundred-something-years-old water tower very often while commuting via Ostkreuz. I tried replicating every little detail, including the black bricks.

  3. Haus Der Statistik, 2020

    Before living in Berlin, I had only seen this building in pictures. The first time I saw it in person I wasn't looking for it. It's a very contrasting view right in the middle of the city.

  4. Berliner Strassenbahn, 2019

    One of the things I enjoy the most in Berlin is the extensive options of public transportation. I've never lived in a city with trams, so for me riding one is always a pleasant novelty.

  5. Bell's Air Taxi, 2019

    This year the company Bell showed Nexus at CES 2019, an air taxi concept. This is my isometric representation of the hybrid-electric flying car that intends to be available via Uber.

  6. Vespa Helmet, 2018

    Another week, another challenge on #spriteweekly. I practiced some lights and shadows on an italian vespa-like helmet in 32x32. It was kind of tricky to work with a spherical shape.

  7. Guy Hawkes Mask, 2018

    I struggled working on a 16x16 grid due to the limited space and available colors. But, for those reasons, I really enjoyed working within the constraints for #spriteweekly.

  8. Rolleiflex, 2018

    A beautiful german twin-lens reflex. I made a frontal drawing instead of the usual isometric that I do, to keep it simple. This is my entry for the #black challenge of #spriteweekly.

  9. Toyota e-Palette, 2018

    Shown at CES 2018 in Las Vegas, the e-Palette is a fully-automated, electric, flexible vehicle that can be tailored to various needs such as parcel delivery, and on-the-road e-commerce.

  10. Telephone Kiosk, 2018

    I found out that the traditional red telephone kiosk from UK has different versions. I picked the K6 (kiosk number six) of 1936 for the second challenge of #spriteweekly #red.

  11. Raspberry Pi Model 1, 2018

    First pixelart drawing after more than a decade. Feels so good to try it again.
