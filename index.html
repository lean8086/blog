Code

Collection of quick experiments and unfinished projects.

  1. CSS-Only 3D Sphere

    The perception of volume is created by stacking 21 different elements separated by equal distances. (Feb, 2021)

  2. Spell checker

    After counting words from 20 popular books, it suggests the nearest ones to replace spelling mistakes. (Nov, 2020)

  3. Wunderpaint

    Pixel Art editor in vanilla JS and functional progr. A canvas is used *only* to process i/o. The user interacts with a base64 image. (Oct, 2020)

  4. Punchy

    WebMIDI and WebAudio let users play a MIDI controller on a synthesizer or sampler directly in the browser. (May, 2020)

  5. Feeds Aggregator

    Reads RSS feeds using a serverless function and normalizes the output to be displayed in a list. (Nov, 2019)

  6. CSS Voxels

    One HTML element per voxel in a matrix. Voxels are arranged on the x/y coordinates and stacked in different layers on the Z-axis. (Feb, 2019)

  7. Single Element Cube

    The element and pseudo-elements are the faces. When it rotates, it changes the transform-origin to always display 3 faces. (Feb, 2019)

  8. Single Element Logo

    Playing with shapes as borders, backgrounds, and characters as part of the content of the pseudo-elements. (Sep, 2018)

  9. CSS Retro Calculator

    A fully working Casio calculator built with Create React App and reducers to update the state. The visuals are all CSS. (Sep, 2018)

  10. Human-readable Filters

    Conceptual experiment to minimize the humans' learning curve for filtering lists. (Jun, 2018)

  11. Autumn Leaf

    An emoji flattened with the text-shadow technique, overrotating in a couple of axes. (Jun, 2018)

  12. Mobile Game Patterns

    Collection of mobile games design patterns for the inspiration of developers, designers, and artists. (Feb, 2013)

  13. Circle Progress Bar

    Handling an SVG path from one single variable to show a wheel indicator. (Oct, 2012)

  14. Time is Ticking

    Modifying the new CSS property transform-origin to distribute letters around a circle. (Oct, 2012)

  15. Doodle Lander

    Playing with artificial gravity on canvas. I used to play Moon Lander and wanted to recreate some of the physics in JavaScript. (Oct, 2012)

  16. 70 Seconds of Terror

    Interactive infographic about the landing of Curiosity Mars Rover, in 13 kB including vector graphics. (Sep, 2012)

  17. Doodle Car

    My first experiment using canvas. It plays with forces like acceleration, inertia, momentum, etc. (May, 2012)

  18. Isometric Walker

    DOM-based matrix of tiles rotated in CSS. When clicking on a tile, an algorithm calculates the shortest path (max. 2 directions). (Jul, 2011)

  19. SkiFree

    DOM-based recreation of the game for Windows. The DOM elements are destroyed every time they left the viewport. (Feb, 2011)

  20. Just Fruits

    My first fully working experiment. It required so much processing to walk that the pace depended on the browser. (Aug, 2009)

Things

Objects that combine hardware and software using everyday materials and tools.

  1. Word Clock

    I found a vintage clock on the street and upcycled it into a word clock using an individually addressable LED strip and Arduino. (Feb, 2021)

  2. Tesla Roadster At Scale

    3D printed parts are rough, and I found the perfect project to learn as much as possible about how to post-process them. (Nov, 2020)

  3. MIDI Controller

    I built this as a minimum viable product for beginners. It has no knobs, sliders, or faders, and it runs with minimal code. (Jun, 2020)

  4. UV Light Box

    This simple box can make cyanotypes, a photographic printing process that produces blueprints using artificial sunlight. (May, 2020)

  5. Arcade Machine

    I put together an old Raspberry Pi, monitor, and speakers with MDF boards, Sanwa-like joysticks, and HAPP push buttons. (Dec, 2018)
